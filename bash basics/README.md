# Day 15 - Bash Basics

## Commands Learned

### pwd
Print current working directory.

```bash
pwd
```

### ls
List files and directories.

```bash
ls
```

### mkdir
Create a new directory.

```bash
mkdir day15
```

### cd
Change directory.

```bash
cd day15
```

### touch
Create an empty file.

```bash
touch genes.txt
```

### cat
Display file contents.

```bash
cat genes.txt
```

### cp
Copy files.

```bash
cp genes.txt backup.txt
```

### mv
Move or rename files.

```bash
mv backup.txt old_backup.txt
```

### head
Display first lines of a file.

```bash
head genes.txt
```

### tail
Display last lines of a file.

```bash
tail genes.txt
```
